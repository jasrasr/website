About this setup

This blog is a tiny flat-file CMS without the “C” and the “M” and most of the “S”. Each post is a JSON file in /posts. The homepage reads an index JSON.

  • No database
  • No PHP required for viewing (PHP used only for server-side logging)
  • Works great with simple hosting

If you can copy files and commit to Git, you can run this blog.
