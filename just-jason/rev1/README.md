# Just Jason Jamboree Junction

Static website inspired by the repeated-name gag from the screenshots, but built as an original parody-style page using only `Jason`-themed text.

## Files

```text
infinite-jason/
├── index.html
├── assets/
│   ├── css/
│   │   └── style.css
│   └── js/
│       └── script.js
├── .gitignore
└── README.md
```

## Upload

Upload the full contents of this folder to your web host. Example paths:

```text
/public_html/infinite-jason/
/public_html/github/infinite-jason/
```

Then browse to:

```text
https://jasr.me/infinite-jason/
```

or whatever folder you used.
